# AAL

Code and dataset of our paper "[Aspect Aware Learning for Aspect Category Sentiment Analysis](https://dl.acm.org/citation.cfm?id=3350487)" accepted by TKDD 2019.

# Requirements
* python 3.6
* tensorflow 1.3.0
* numpy 1.16.4
* scikit-learn 0.21.2
